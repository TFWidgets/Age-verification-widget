Usage

With Local Config:

<script type="application/json" id="bhw-age-local-config">
{
    "title": "Age Verification",
    "message": "You must be 18+ to continue",
    "frequency": "30d"
}
</script>
<script src="https://your-domain.com/embed.js" data-id="local"></script>

Configuration Options

  • title - Main title text
  • subtitle - Subtitle text
  • message - Main message body
  • frequency - How often to show: 'always', 'session', '24h', '30d'
  • redirectUrl - Where to redirect on decline
  • blockContent - Block page scrolling until verified
  • showDelay - Delay before showing (ms)
